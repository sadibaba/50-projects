## 🎯 Project Overview

**TaskManager** is a comprehensive, modern task management application built with React 19 that combines intelligent planning, scheduling, and productivity tracking in one beautiful interface. This project was collaboratively developed with the incredible assistance of **DeepSeek AI** 🤝.

## ✨ Key Features

### 📋 **Core Planning System**
- **Multi-board Kanban View** (Plan, Schedule, In-Progress, Review, Done, Blocked)
- **Drag & Drop Task Management** with @hello-pangea/dnd
- **Priority-based Task Sorting** (High/Medium/Low)
- **Smart Date Scheduling** with calendar integration
- **Time-based Task Allocation**

### 🎨 **User Profile Management** *(Added Feature)*
- **Complete Profile Dashboard** with editable information
- **LocalStorage-based User Data** (No backend required)
- **Profile Picture Avatar** with user initials
- **Personal Information Management** (Name, Email, Phone, Location)
- **Bio Section** for personal description

### 📊 **Advanced Features**
- **Real-time Statistics Dashboard**
- **Progress Tracking** with visual progress bars
- **Today's Focus** section for daily priorities
- **Upcoming Deadlines** visualization
- **Task Completion Analytics**
- **Priority-based Color Coding**

### 🔒 **Authentication System**
- **JWT-based Login/Register**
- **Protected Routes**
- **LocalStorage Token Management**
- **Secure Password Handling**

### 📱 **UI/UX Excellence**
- **Fully Responsive Design** (Mobile, Tablet, Desktop)
- **Gradient Backgrounds** with modern aesthetics
- **Smooth Animations & Transitions**
- **Intuitive Icon Navigation**
- **Toast Notifications** for user feedback
- **Dark/Light Mode Ready**

## 🛠️ Technology Stack

### **Frontend**
- ⚛️ **React 19** (Latest version)
- 🎨 **Tailwind CSS** for styling
- 🔄 **@hello-pangea/dnd** for drag & drop
- 📅 **Lucide React** for icons
- 🚀 **React Router** for navigation
- 🔧 **Axios** for API calls

### **Backend Integration Ready**
- 🔐 **JWT Authentication**
- 📡 **RESTful API Structure**
- 💾 **LocalStorage Fallback**
- 🔄 **Real-time Updates**

## 🚀 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/sadibaba/50-projects/tree/main/Phase-2/1-TaskManager

# Navigate to project directory
cd backend || frontend

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
